🚀 快速開始（3 步驟）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步驟 1：安裝系統
  $ ./install.sh
  或手動：
  $ pip install -r requirements.txt
  $ cp config/settings.example.json config/settings.json

步驟 2：設定 API
  $ python3 scripts/setup.py
  或手動編輯：
  $ nano config/settings.json
  # 填入永豐 API Key 和 Secret Key

步驟 3：測試運行
  $ python3 scripts/scanner.py --format text
  或使用啟動選單：
  $ ./start.sh
