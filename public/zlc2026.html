📘 第四章｜危機模型：2026 危機指數（Crisis Score）

危機指數（Crisis Score, CS）為 APP 的核心計算機制，會從 0 ～ 100 分動態變化。
整體由 6 大類變數組成。

  • 1. 黨內團結度（IU：Internal Unity）
    反映「本土派 / 深藍派 / 地方派」對主席的服從程度。
  • 2. 媒體輿論指數（MO：Media Opinion）
    依電視台、報紙、社群情緒進行即時分析。
  • 3. 民調溫度（PT：Poll Temperature）
    追蹤主席個人好感、政黨支持度、泛藍整體投票意願。
  • 4. 危機事件（CE：Crisis Events）
    例如：失言、提名爭議、地方派系反彈、外部攻擊。
  • 5. 中央執委穩定度（CC：Central Committee Stability）
    判斷是否有「臨時全代會」風險。
  • 6. 2026 選戰壓力（EP：Election Pressure）
    反映六都與關鍵縣市的勝選機率與派系焦慮。

最終計算公式： 

CS = 0.22*IU + 0.18*MO + 0.15*PT + 0.25*CE + 0.10*CC + 0.10*EP

當 CS > 70 時，系統會標記：⚠️「主席不穩定」
當 CS > 85 時，系統會標記：🔥「可能啟動換主席程序」

📘 第五章｜2026 危機劇情（6 條分岐線）

APP 的劇情模擬依照計算出的 CS（危機指數）動態生成。

  1. 1. 低風險分支（CS < 40）
    描述鄭主席鞏固深藍、成功安撫地方派系，民調止跌回升。
  2. 2. 中風險分支（40 ≤ CS < 60）
    媒體輕微攻擊、黨內傳出雜音但仍可控制。
  3. 3. 高風險分支（60 ≤ CS < 75）
    地方派系開始串聯，出現「2026 一輸主席就下台」的壓力訴求。
  4. 4. 臨界分支（75 ≤ CS < 85）
    某一都會縣市提名案失敗 → 爆發黨內公開衝突。
  5. 5. 臨時全代會分支（85 ≤ CS < 92）
    部分中央委員開始蒐集連署，暗示主席可能被「提前換掉」。
  6. 6. 換主席分支（CS ≥ 92）
    劇情引導至「是否啟動臨時全代會？」與「接任者：？」的 AI 推演。
